# AI助手文件上传功能说明

## 功能概述

AI助手现已支持文件上传功能，用户可以上传图片和文档与AI进行交互。

## 新增功能

### 🎯 界面改进
- **重新设计的输入区域**：将模型选择和文件上传按钮整合到消息输入框
- **工具栏布局**：模型选择和文件上传按钮位于输入框左下侧
- **API设置按钮**：移至输入框右上角，更加便捷

### 📎 文件上传功能
- **支持的文件类型**：
  - 图片：jpg, png, gif, webp, svg等
  - 文档：pdf, doc, docx, txt, md, json, csv, xlsx, xls
- **文件大小限制**：单个文件最大10MB
- **多文件支持**：可同时上传多个文件

### 🖼️ 文件预览
- **图片预览**：上传后即时显示缩略图
- **文档图标**：文档文件显示统一图标和文件名
- **文件信息**：显示文件名和大小
- **移除功能**：可单独移除或批量清空文件

### 💬 消息显示
- **附件展示**：消息中显示上传的文件
- **图片查看**：点击图片可全屏预览
- **文档下载**：点击文档可下载查看
- **文件信息**：显示文件名和大小

## 使用方法

### 1. 上传文件
1. 点击输入框左下角的文件上传按钮（📎图标）
2. 选择要上传的文件（支持多选）
3. 文件将显示在输入框上方的预览区域

### 2. 管理文件
- **移除单个文件**：点击文件预览右上角的×按钮
- **清空所有文件**：点击预览区域右上角的"清空"按钮
- **查看文件信息**：文件名和大小显示在预览下方

### 3. 发送消息
- 可以只发送文件（不输入文字）
- 可以同时发送文字和文件
- 发送后文件会自动清空

### 4. 查看历史消息
- 图片：点击可全屏预览
- 文档：点击下载按钮可下载文件
- 文件信息：显示在消息气泡中

## 技术特性

### 🔒 安全性
- 文件仅在本地处理，不会自动上传到服务器
- 支持的文件类型经过严格限制
- 文件大小限制防止内存溢出

### 🎨 用户体验
- 拖拽式文件预览界面
- 平滑的动画效果
- 响应式设计，适配不同屏幕尺寸
- 直观的文件类型图标

### 🤖 AI模型支持
- **多模态模型**：Gemini、ChatGPT等支持图片理解
- **文本模型**：其他模型会提示文件信息但不处理内容
- **扩展性**：为未来的多模态功能预留接口

## 注意事项

1. **文件大小限制**：单个文件不能超过10MB
2. **模型兼容性**：不是所有AI模型都支持文件处理
3. **网络传输**：大文件可能影响响应速度
4. **存储空间**：文件会临时存储在本地内存中
